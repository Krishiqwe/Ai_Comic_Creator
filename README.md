# Ai_Comic_Creator
# 🎨 AI Comic Creator

An AI-powered full-stack web application that transforms your ideas into illustrated comic strips — complete with generated scripts and panel artwork.

---

## ✨ Features

- 📝 **AI Script Generation** — Powered by Groq's `llama-3.3-70b-versatile` model to craft witty, structured comic narratives from a simple prompt
- 🖼️ **Automatic Panel Images** — Each comic panel is illustrated using Pollinations.ai's image generation API
- ⚡ **Fast Inference** — Groq's ultra-low latency API ensures near-instant script generation
- 🌐 **Web Interface** — Clean Flask-based frontend to input prompts and view the generated comic

---

## 🛠️ Tech Stack

| Layer | Technology |
|---|---|
| Backend | Python, Flask |
| AI / LLM | Groq API (`llama-3.3-70b-versatile`) |
| Image Generation | Pollinations.ai |
| Frontend | HTML, CSS, JavaScript |
